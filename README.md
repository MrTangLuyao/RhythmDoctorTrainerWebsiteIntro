# Rhythm Doctor Trainer — 安装引导页

[节奏医生修改器](https://github.com/Cohenjikan/RhythmDoctorTrainer)（Windows）与
[macOS 版](https://github.com/MrTangLuyao/RhythmDoctorTrainerMac)的玩家向安装引导页。

- 首页只有两个按钮：Windows 安装（蓝）/ macOS 安装（白），点击滑出对应的三步安装页
- Windows 走 v2.2.0 的一键安装器（INSTALL.bat），macOS 走一行终端命令（带复制按钮）
- 双语（按设备语言列表自动检测，可手动切换）
- 纯静态零构建零依赖，任意静态托管直接部署
